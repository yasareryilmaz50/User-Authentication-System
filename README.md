User Authentication System

C# ve .NET kullanılarak geliştirilmiş console tabanlı bir kullanıcı kayıt ve giriş (authentication) uygulamasıdır.
Bu proje, temel kullanıcı yönetimi ve kimlik doğrulama mantığını uygulamalı olarak göstermektedir.

Projenin Amacı

Bu proje;

C# dilinde Nesne Tabanlı Programlama (OOP) yapısını uygulamak

Liste tabanlı kullanıcı yönetimi yapmak

LINQ sorgularını kullanmak

Temel kimlik doğrulama (authentication) mantığını anlamak

Backend düşünce yapısını geliştirmek

amacıyla geliştirilmiştir.
